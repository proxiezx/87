About SwitchyOmega

Version 2.5.20

SwitchyOmega is a proxy manager and switcher that helps you manage and switch between multiple proxies quickly and easily.

Features

  • Quickly switch between different proxy configurations
  • Support for HTTP, HTTPS, SOCKS4, and SOCKS5 proxies
  • Site-specific proxy rules
  • Import and export settings

For more information, visit the GitHub repository.
